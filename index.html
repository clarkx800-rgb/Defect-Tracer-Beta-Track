SYSTEM INITIALIZED

STEP 1: ESTABLISH ROUTE (In Order)

  • Open the top right menu.
  • Select 1. LINE, 2. DIRECTION (IB or OB), and 3. SUB-TO-SUB AREA.
  • Tap INITIATE SECTIONS to generate your inspection route.

STEP 2: LOG DEFECTS

  • Scroll to current Track Section (TS).
  • Select COMPONENT and DEFECT from dropdowns.
  • Tap 📷 PHOTO (if needed) to attach visual evidence or start reference of progressing defect.

STEP 3: SAVE DATA (not export) ⚠️

  • System does not auto-save. Open menu and tap 💾 SAVE DATA frequently incase phone crash or browser accidental refresh.
  • If system restarts, tap 📂 LOAD DATA and load saved file to resume.
